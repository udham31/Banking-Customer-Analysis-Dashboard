# 🏦 Banking Customer Analysis Dashboard

## 📌 Project Overview

This project is an end-to-end Banking Customer Analytics solution built using **SQL, PostgreSQL, Excel, and Power BI**. It focuses on analyzing customer, loan, and transaction data to generate business insights through interactive dashboards and SQL-based analysis.

---

## 🎯 Business Objectives

- Analyze customer demographics and behavior.
- Evaluate loan portfolio performance.
- Monitor transaction trends.
- Identify high-value customers.
- Generate business KPIs through interactive dashboards.

---
